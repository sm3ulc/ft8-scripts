## Usage

Edit splitter.py and replace ramdrive with a suitable directory

```
nano splitter.oy
```

To start dumping IQ-data into files with a RTL-device

```
rtl_sdr -d 0 -g 10 -s 1200000 -f 10000000 -D 2 - | python3 splitter.py -b 30
```
